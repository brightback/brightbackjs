Brightback Install

1. Include the brightback js in the head element of your page

<script type="text/javascript" src="https://app.brightback.com/js/current/?compiled=true"></script>

2. Create a cancel button

<a id="bb-cancel" href="/fallback">Cancel</a>

3. Run the brightback js on page load. This code should be placed before the closing body tag of your page

<script type="text/javascript">
  if (window.Brightback) {
    window.Brightback.handleData({
      app_id: 'APP_ID',
      email: 'jdoe@example.com',
      account: {
       internal_id: '1234AZ55'
      }
    });
  }
</script>
